ZN004-CH344Q-四电压
===================

产品介绍

ZN004-CH344Q为一款四路USB转TTL工具板，串口电压以及输出电压支持四电压，即1.8V、2.5V、3.3V、5V。通过USB方口与PC端进行连接，默认配套上盖亚克力，对外接口为XH2.54-4P端子

因对外端子接口不是每个客户需要的，默认不赠送XH2.54-4P的线材，需客户根据自身需求，进行选购！
